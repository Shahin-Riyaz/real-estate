# MERN Real Estate Platform 🏠

A full-stack real estate web application built with the MERN stack (MongoDB, Express, React, Node.js) featuring:

- 🧠 AI-powered chatbot 
- 🔐 Authentication with Firebase
- 🏡 Create, update, delete listings
- 🔍 Search and filter listings
- 📦 Upload images with Cloudinary

## 🚀 Getting Started

##Backend

cd api
npm install
npm run dev

##Frontend

cd client
npm install
npm run dev
